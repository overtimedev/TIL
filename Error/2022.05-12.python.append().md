## 파이썬 append() Error
**Type Error : append()**
Python append()함수에서 아래와 같은 오류가 발생
오류에 대해 찾으면 알게된 점
> append()함수는 하나의 원소밖에 넘겨주지 못함.
> 15번째 줄에 direction, length를 () (괄호)로 묶으면 해결된다.
```pytohn
  def func(x):
    print(x)
func(1, 2)
```
이와 같은 코드에서도 같은 오류가 발생한다.
